# Scripts de Conversão MD → ODT/DOCX

Scripts Python para converter relatórios técnicos Markdown para formatos de documentos editáveis (ODT e DOCX) com renderização de diagramas Mermaid.

## 📋 Scripts Disponíveis

| Script | Formato | Uso Principal |
|--------|---------|---------------|
| `convert_md_to_odt.py` | ODT (LibreOffice) | Linux, compatibilidade OpenDocument |
| `convert_md_to_docx.py` | DOCX (MS Word) | Windows, ambiente corporativo |

## 🚀 Uso Rápido via Skills

### Converter para ODT
\`\`\`bash
/convert-md-to-odt                    # Lista e escolhe arquivo
/convert-md-to-odt relatorio.md       # Converte arquivo específico
/convert-md-to-odt --all              # Converte todos
\`\`\`

### Converter para DOCX
\`\`\`bash
/convert-md-to-docx                   # Lista e escolhe arquivo
/convert-md-to-docx relatorio.md      # Converte arquivo específico
/convert-md-to-docx --all             # Converte todos
\`\`\`
