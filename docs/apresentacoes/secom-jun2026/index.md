## Os 6 slides

| # | Tema | Foco |
|---|------|------|
| 01 | Capa | "Da notícia ao dado: infraestrutura de inteligência para a comunicação do governo federal" |
| 02 | Acervo do DGB | Engenharia de Dados — 160+ portais, 300 mil+ notícias, ~5 min de latência, +25 anos de histórico |
| 03 | IA para Enriquecer os Dados | Ciência de Dados + Ciência da Informação — classificação temática, NER, embeddings, 4 BDs complementares |
| 04 | Ferramentas e Serviços | Web Difusora · Clipping · Panorama Gov.BR — pacote para ASCOMs |
| 05 | API + MCP + Agente de IA | Camada de integração programática — GraphQL, MCP Server (piloto ativo), chat conversacional |
| 06 | Futuro | DOU · outros poderes · esferas subnacionais — "a memória da democracia brasileira" |
